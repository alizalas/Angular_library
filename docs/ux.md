# UX-концепция BookShelf

## Целевая аудитория

### Наши пользователи - современные, практичные люди, любящие читать книги

### Персона 1: Анна, 28 лет
**Бэкграунд:** Маркетолог, любит читать бизнес-литературу и нон-фикшн

**Цели:**
- Систематизировать прочитанные книги для работы
- Сохранять ключевые идеи и цитаты
- Отслеживать профессиональное развитие через чтение
- Делиться полезными книгами с коллегами

**Ожидания:**
- Быстрый поиск по книгам
- Удобное добавление заметок
- Профессиональный минималистичный дизайн
- Возможность экспорта заметок

### Персона 2: Михаил, 22 года
**Бэкграунд:** Студент, увлекается фантастикой и фэнтези

**Цели:**
- Вести учет прочитанных книг
- Получать рекомендации на основе предпочтений
- Обмениваться списками с друзьями
- Отслеживать свои читательские цели

**Ожидания:**
- Геймификация (цели по чтению)
- Социальные функции
- Красивые обложки книг
- Мобильная версия для чтения в транспорте

## Ключевые сценарии (User Stories)

### MVP (первый экран)
1. **AS IS** пользователь, **I WANT** видеть свою библиотеку с фильтрацией по статусу, **SO THAT** быстро находить нужные книги
2. **AS IS** пользователь, **I WANT** добавлять новую книгу через форму с автозаполнением, **SO THAT** экономить время на вводе данных
3. **AS IS** пользователь, **I WANT** видеть статистику чтения на дашборде, **SO THAT** отслеживать свой прогресс

### Вторичные сценарии
4. **AS IS** пользователь, **I WANT** создавать заметки и сохранять цитаты, **SO THAT** не потерять важные мысли
5. **AS IS** пользователь, **I WANT** делиться списком рекомендаций, **SO THAT** советовать книги друзьям

## Дизайн-система (Taiga UI 4)

### Компоненты Taiga UI для использования:
- **Navigation:** TuiTabs, TuiSidebar
- **Forms:** TuiInput, TuiTextarea, TuiSelect, TuiDataList
- **Cards:** TuiCard, TuiIsland
- **Feedback:** TuiRating, TuiProgress
- **Data Display:** TuiTable, TuiBadge, TuiTag
- **Overlays:** TuiDialog, TuiNotification
- **Buttons:** TuiButton, TuiIconButton

### Цветовая схема:
- Основной: Taiga UI default theme
- Акцентный: #526ed3 (книжный синий)
- Фоны: светлые, минималистичные

### Типография:
- Заголовки: системный шрифт, жирный
- Основной текст: 16px, comfortable reading
